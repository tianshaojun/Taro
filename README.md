### 1.Taro框架简介

#### Taro支持的终端

+ 微信小程序
+ H5移动端Web页面
+ 百度小程序
+ 支付宝小程序
+ 快应用
+ ReactNative
+ 字节跳动小程序
+ QQ轻应用

### 2.Taro开发环境的搭建

+ Taro开发环境的安装
   1.npm的安装方式  npm install -g @tarojs/cli
   2.yarn的安装方式  yarn global add @tarojs/cli
+ 用脚手架创建项目  taro init myDemo1
+ 启动项目  yarn dev:h5

### 3.Taro生成小程序代码

+ yarn dev:weapp
